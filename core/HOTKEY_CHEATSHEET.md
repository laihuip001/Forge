# Forge Hotkey チートシート

> **Version:** 1.0 | **Date:** 2026-01-14

---

## Essential (必須: 5個)

最初に覚えるべきコマンド。

| Hotkey | 機能 | 用途 |
|:------:|------|------|
| `[Plan]` | 計画モード | 実装計画Artifactを生成、承認待ち |
| `[Act]` | 実行モード | 承認済み計画を実行、Diff生成に集中 |
| `[F]` | Fork/Export | 変数解決済みの最終出力、コピペ可能 |
| `[!]` | 強制実行 | Mode Aを強制、ダッシュボード省略 |
| `[Audit]` | 監査モード | Pre-Turn Audit有効化 |

---

## Extended (拡張: 8個)

必要に応じて使用。

| Hotkey | 機能 | 用途 |
|:------:|------|------|
| `[Verify]` | 検証モード | テスト/Lint/ブラウザ検証 |
| `[Deep]` | 深掘り | 2次/3次影響まで推論拡張 |
| `[Report]` | Full Disclosure | 隠れた推論ログを表示 |
| `[Quick]` | 即時確定 | レビュースキップで `[F]` 実行 |
| `[C]` | To Claude | Claude用XMLに変換 |
| `[G]` | To GPT | ChatGPT用Markdownに変換 |
| `[T]` | Test Case | 3シナリオのテスト生成 |
| `[Audit Full]` | 常時監査 | Pre-Turn Audit常時有効化 |

---

## ワークフロー用 (prompt-development)

| Hotkey | 機能 |
|:------:|------|
| `[New]` | 新規プロンプト開発開始 |
| `[Req]` | 要件定義フェーズへ |
| `[Design]` | 設計フェーズへ |
| `[Impl]` | 実装フェーズへ |
| `[QA]` | 品質検証フェーズへ |
| `/archive` | アーカイブに保存 |

---

**Tip:** `[!]` + リクエスト = 最速でコード/プロンプト出力
